Η επίσκεψη Μέρκελ στην Αθήνα ερμηνεύθηκε εξαρχής, για προφανείς λόγους, ως μία κίνηση υψηλού συμβολισμού, από αυτές που φέρνουν νέα δεδομένα στη σκακιέρα της ευρωπαϊκής κρίσης. Η λέξη "μήνυμα" πρέπει να γράφτηκε και να μεταδόθηκε μερικές χιλιάδες φορές την περασμένη εβδομάδα. Σχεδόν όλες οι αναλύσεις στα ελληνικά ΜΜΕ, όμως, έπαιρναν ως δεδομένο πως η Μέρκελ ήρθε εδώ, γιατί ήθελε να δείξει και να πει κάτι στους Έλληνες. Είναι η μισή αλήθεια. Γιατί κάθε ηγέτης μεγάλης χώρας που ταξιδεύει στο εξωτερικό, γνωρίζει ότι αμέσως μετά θα επιστρέψει. Γνωρίζει, επίσης, ποιό είναι το κλίμα που θα συναντήσει εκεί που πηγαίνει. Παραμονές της επίσκεψης, οι Γερμανοί δημοσιογράφοι περιέγραφαν την επίσκεψη Μέρκελ, ως "ειδική αποστολή" στην Καμπούλ...

Ας δούμε, λοιπόν, για λίγο αυτή την επίσκεψη από την ανάποδη. Από όλα όσα είπε κι έκανε στην Αθήνα η Μέρκελ, το γερμανικό πρακτορείο ειδήσεων (DPA) στάθηκε σε μία φράση της και εκεί στήριξε το τηλεγράφημά του για τα αποτελέσματα της επίσκεψης: για πρώτη φορά η Καγκελάριος συνέκρινε σε δηλώσεις της την Ελλάδα με την πρώην Ανατολική Γερμανία, έγραψε ο απεσταλμένος του DPA στην Αθήνα (9 Οκτωβρίου 2102), επισημαίνοντας ότι η αναφορά Μέρκελ στα 23 χρόνια που μεσολάβησαν από την επανένωση της χώρας, αποτελεί ένα παράδειγμα εύκολα κατανοητό από κάθε Γερμανό, που δυσπιστεί μπροστά στη δυσκολία να υλοποιηθούν μεταρρυθμίσεις στην Ελλάδα.

Τρεις ημέρες αργότερα, την περασμένη Παρασκευή (12 Οκτωβρίου) η Frankfurter Allgemeine Zeitung έγραφε πως η Μέρκελ "ετοιμάζει το έδαφος για να πει την αλήθεια για την Ελλάδα στους Γερμανούς", εξηγώντας ότι πλησιάζει η ώρα που η Καγκελάριος θα ζητήσει από τους συμπολίτες της να βοηθήσουν περισσότερο τους Έλληνες.

Η Μέρκελ ήρθε γνωρίζοντας ότι θα μεταδοθούν στη Γερμανία και σε όλη την Ευρώπη οι εικόνες των διαδηλωτών στο Σύνταγμα. Και ήρθε γιατί ήθελε να μιλήσει στους Γερμανούς από εδώ, ώστε να υποχρεωθούν να προσέξουν περισσότερο αυτά που είχε να τους πει - με φόντο Έλληνες που κατηγορούν τη Γερμανία (όχι απαραίτητα δικαίως σε όλες τις περιπτώσεις, αν όχι άδικα στις περισσότερες) για όσα περνούν. Προσωπικά, πίστευα ότι αυτό είχε στο μυαλό της η Μέρκελ, από την ώρα που ανακοινώθηκε η επίσκεψη. Η Καγκελάριος πήρε απόφαση να στρίψει το καράβι και το εγχείρημα βρίσκεται σε εξέλιξη. Πλησιάζει η ώρα των αποφάσεων. Στα σχέδια της Μέρκελ - τώρα πια είναι ξεκάθαρο - υπάρχει και η Ελλάδα στην ευρωζώνη, υπό τον όρο ότι θα κάνει την προσπάθεια να καλύψει ένα μέρος από την απόσταση που της αναλογεί. Η Γερμανία συζητά να καλύψει την υπόλοιπη...

Πρώτα, όμως, πρέπει να πάρουν τις δικές τους αποφάσεις η κυβέρνηση και τα κόμματα, εδώ στην Αθήνα. Δε θα είναι καθόλου, μα καθόλου εύκολες. Εδώ και μερικές ώρες, ακούμε για «βραχυκύκλωμα» στις διαπραγματεύσεις της Τρόικας με υπουργούς για τις απολύσεις στο Δημόσιο (οι Τροϊκανοί δεν δέχονται τη θεωρία, ότι αντί να απολυθούν θα «αξιοποιηθούν» οι δημόσιοι υπάλληλοι από οργανισμούς που καταργούνται), καθώς και για ορισμένες από τις επώδυνες αλλαγές στα εργασιακά. Αυτό σημαίνει ότι τις αμέσως επόμενες ώρες ή 24ωρα, οι υπουργοί θα πετάξουν όλες τις «καυτές πατάτες» στα χέρια των αρχηγών για να πάρουν εκείνοι τις δύσκολες αποφάσεις σε μία σύσκεψη. Ο Σαμαράς θέλει να πάει την Πέμπτη στη σύνοδο κορυφής, στις Βρυξέλλες, με το πακέτο μέτρων κλεισμένο. Είναι εξαιρετικά αμφίβολο, όμως, άν θα είναι σε θέση οι τρεις αρχηγοί να λάβουν τόσο δύσκολες αποφάσεις, τόσο γρήγορα. Και, αν ακόμη οι τρεις συμφωνήσουν, αν θα μπορέσουν να πείσουν τους 179 βουλευτές τους. Βλέπετε, είναι πολύ ευκολότερο να παριστάνεις τον "αντιστεκόμενο αδικημένο", από το να κάνεις πολιτική.

Η Μέρκελ έλαβε τις αποφάσεις της. Το ερώτημα είναι αν η ελληνική κυβέρνηση μπορεί να πάρει τις δικές της. Κυριακή, κοντή γιορτή.
